# fbratten.github.io

Personal landing page for Fredrik Brattén.

**Live Site:** https://fbratten.github.io/

## Tech Stack

- Pure HTML + [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Shields.io](https://shields.io/) badges
- [readme-typing-svg](https://github.com/DenverCoder1/readme-typing-svg) animation
- Hosted on GitHub Pages

## Features

- Dark theme with card-based project grid
- Hover effects and gradient accents
- Responsive mobile layout
- Links to all project showcases and demos
